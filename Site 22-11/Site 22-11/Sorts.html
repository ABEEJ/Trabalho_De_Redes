- 1° Tipo de Ordenação -

- Merge Sort -


Merge sort é um algoritmo de classificação poderoso que divide uma lista em duas partes, classifica cada parte e depois mescla as duas partes classificadas para produzir uma lista classificada. Uma abordagem sistemática fornece desempenho previsível e eficiente para grandes conjuntos de dados.
